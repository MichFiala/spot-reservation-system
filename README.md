# Spot reservation system

## Features

### Page - Readonly

User should see:

- Name of the page
- Description
- Map with available spots
- About segment
- Kontakt segment

User should be able to click on each spot and reserve spot in available time. 
User should be able to click and see terms of condition.




### Admin - Page

Page should 

### User should be able to create page for group of spots with map

On page there should be map with available spots.
Not available spots should be marked with different colors.

### User should be able to group of spots on map

User should be able to create new group of spots and be able to create each spot 

Spot should have these informations:

- Description
- Coordinates
- Avaibility


### Each spot can be reserved

Each spot can be clicked and reservation can happen for spots that are available
When reserving user should agree with terms of condition


### User types

Normal users can see their reservations but cannot create their own page.
Users with certain conditions can create X amount of pages and publish them.
